# Content Factory

Автоматизированное ведение Telegram-канала Игоря Шореца — сомелье и wine director.

## Пайплайн

```mermaid
flowchart TD
    A([💡 Идея]) --> B[Агент оформляет пост\nв posts/YYYY-MM-DD.md]
    B --> C[Статус: draft]
    C --> D{Нужна правка?}
    D -- да --> E[Редактирование]
    E --> D
    D -- нет --> F[Статус: ready]
    F --> G[Публикация в Telegram]
    G --> H[Файл перемещается\nв published/]
    H --> I[Статус: published]

    B --> J[(content_plan.md)]
    F --> J
    I --> J
```

## Структура

```
Content-factory/
├── agents.md           # правила агента
├── content_plan.md     # контент-план со статусами
├── posts/              # черновики и готовые посты
└── published/          # архив опубликованных постов
```

## Статусы поста

| Статус | Описание |
|---|---|
| `draft` | Сырая идея, требует доработки |
| `ready` | Готов к публикации |
| `published` | Опубликован, перемещён в архив |
